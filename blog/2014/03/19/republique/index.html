Republique

Mar 19th, 2014

- Здравствуйте, меня зовут Роман и я люблю игры.
- Здравствуй, Роман.

Иногда в потоке игр возникают брилианты, о которых молчать нельзя. Одним из подобных брилиантов, втретившихся мне в последнее время, является игра Republique. Игра представляет собой нехарактерный для iOS-платформы стелс. Разработчики умудрились сделать стелс-экшн который может управляться с помощью тач управления. С другой стороны эту игру можно отнести к жанру интерактивного кино.

Сюжет рассказывает о мире подобном Оруэловскому 1984, и персонаже не согласным с этим миром. Не борцом с системой, но человеке пытающемся ускользнуть из нее.

Игровой процесс заключается в том что игрок наблюдает за историей через систему видео-наблюдения и дает подсказки главной героине о продвижении вперед. При этом главная героиня управляется AI, и игрок может только дать подсказку куда передвигаться, но не может управлять главным героям напрямую.

Игра планируется выпускаться по частям, с возможностью передвижения между частями назад и вперед. На данный момент доступна только первая часть из пяти, но в ней уже присутствуют места куда возможно попасть только после прохождения следущих.
